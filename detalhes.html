Descrição do Problema

Necessitamos de uma plataforma web responsiva para cadastrar, editar e excluir itens do nosso almoxarifado. Atualmente, o controle é feito em planilhas de papel, o que gera perda de dados e furos de estoque.

O sistema deve possuir alertas visuais (vermelho/amarelo) quando um produto atingir a quantidade mínima de segurança, além de permitir o cadastro de categorias de produtos.

Requisitos Técnicos Esperados

  • Interface construída em HTML, CSS e JavaScript.
  • Banco de Dados relacional (SQL Server).
  • Operações completas de CRUD (Create, Read, Update, Delete).
